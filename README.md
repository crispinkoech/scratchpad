# What to get involved?
## Project setup
```
git clone https://github.com/j-000/scratchpad.git
npm install
npm run serve
```
Now do some coding!

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
